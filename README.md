# StudyBuddyApp : Mpbilprogrammering ITF21019
### Gruppe 22: Mikkel & Shahd

StudyBuddy er en mobilapplikasjon laget med **React Native (Expo)** og **Firebase**, utviklet for å hjelpe studenter med å strukturere studietiden sin.  
Appen lar brukere legge inn oppgaver, se dem i kalenderen, chatte med gruppemedlemmer og bruke fokusmodus for bedre konsentrasjon.

---

## Forutsetninger
Før du starter, sørg for at du har installert:
- Node.js (versjon 16 eller nyere)  
- npm (følger med Node.js)  
- Git  
- Expo Go-appen (på mobil) eller en moderne nettleser (Chrome/Edge)  

---

## Installasjon

### Klon prosjektet:
```bash
git clone https://github.com/shmudi/StudyBuddyApp_Gruppe22.git
cd StudyBuddyApp_Gruppe22
```

### Avhengigheter:
``` bash
npm install
```

## Start prosjektet:
```bash
npx expo start
```
